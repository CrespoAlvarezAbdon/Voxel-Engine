 Set the crossover operator's split point. It is unused in implementation 1 of the crossover operator. Example: if we have a neural network with 5 hidden layers and the split point is set at point 2, the weights that connect the input layer (layer 0) to the first hidden layer (layer 1), the ones that connect the first hidden layer to the second hidden layer (layer 2) and the ones that connect the second hidden layer to the third hidden layer (layer 3) will be exchanged between the parents to create the new offspring.
