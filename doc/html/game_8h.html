class  VoxelEng::game
 Game engine API responsible for all the basic engines operations (startup, menu/level/AI mode loops, access to save slots...). This class also serves as a way to abstract classes that directly implement the engine. For example, instead of storing a reference to the player camera for the engines' graphical mode and polluting the code, said camera can be accessed by calling game::playerCamera(). More...
