 Set the block texture atlas' resolution per block. The resolution will always make block textures be square. That is, if you execute texture::blockAtlasResolution() = 32, now the block textures will have a resolution of 32x32 pixels. WARNING. You must set the resolution at least once before the method chunkManager::manageChunks starts executing in the chunk management thread.
